### SmartPark — Plataforma SaaS de Gestao de Estacionamento

> A matricula e o bilhete. O telemovel e o pagamento. A cancela abre sozinha.

Projeto de Prova de Aptidao Profissional. Sistema completo ponta a ponta: reconhecimento automatico de matriculas via computer vision, dashboard multi-parque em tempo real, PWA de pagamento self-service com seis metodos via Stripe, e controlo de hardware Arduino para abertura de cancelas.

**Stack:** React · TypeScript · Node.js · FastAPI · Python · OpenCV · MariaDB · Docker · Stripe Connect · Arduino

**[Repositorio showcase](https://github.com/Shadoww111/smartpark-saas)** · **[Landing Page](https://landingpage-sp.ruicosta.pt)** · **[Dashboard](https://dashboard-sp.ruicosta.pt)** · **[PWA Condutor](https://pay-sp.ruicosta.pt/?park=7)**
